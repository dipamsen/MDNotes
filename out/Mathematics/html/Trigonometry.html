Trigonometry

Degree Measurement of Angles

Usually, we measure angles in degrees, where \gdef\cosec{\mathrm{cosec}}360\degree is one complete rotation.

1\degree = 60' 1' = 60''

Radian Measurement of Angles

Radians is another unit for measuring angles, where a complete rotation is marked as 2\pi radians. 360\degree=2\pi

  • A unit circle is a circle with radius 1 unit.
  • The unit ‘radian’ is often omitted while writing radian angles.

Definition

The angle subtended at the centre by an arc of length x in a unit circle is x rad.

\boxed{\text{Radian measure} = \frac{\text{Arc length}}{\text{Radius}}}

Degree-Radian Conversion

180\degree = \pi 1 \text{ radian} = \frac {180\degree}\pi=57\degree 16' 1\degree = \frac \pi{180}=0.01746 \text{ radian}

\boxed{\text{Radian measure} = \frac\pi{180}\times \text{Degree measure}} \boxed{\text{Degree measure} = \frac{180}\pi\times \text{Radian measure}}

Value of 0

\cos\theta is 0 for the values \displaystyle\theta = \frac\pi2, 3\frac\pi2, \cdots

\left.\begin{aligned} \therefore \sin\theta &=0\implies \theta = n\pi\\ \cos\theta &=0\implies \theta = (2n+1)\frac\pi2 \end{aligned}\qquad\right\rbrace n\in Z

Sign of Trigonometric Functions

Negative Angle

From the unit circle, we can determine the value of functions for a negative angle.

We can see that \sin x=b; \cos x=a

For negative value of x (Figure 5), the x coordinate remains unchanged, while the y coordinate becomes negative.

\cos (-x)=\cos x \sin (-x)=-\sin x

Sign in different Quadrants

Quadrant 1: a and b are both positive
Quadrant 2: a is negative and b is positive
Quadrant 3: a and b are both negative
Quadrant 4: a is positive and b is negative

From the signs of \sin x and \cos x, we can find out the signs of all other trigonometric functions.

Trigonometric Identities

  • Negative Angle*
  1. \sin (-a)=-\sin a
  2. \cos (-a)=\cos a
  3. \tan (-a)=-\tan a
  • Sum or Difference of Angles

  1. \sin (a+b) = \sin a\cos b + \cos a\sin b

  2. \sin (a-b) = \sin a\cos b - \cos a\sin b

  3. \cos (a+b) = \cos a\cos b - \sin a\sin b

  4. \cos (a-b) = \cos a\cos b + \sin a\sin b

  5. \displaystyle\tan (a+b) = \frac{\tan a + \tan b}{1-\tan a \tan b}

  6. \displaystyle\tan (a-b) = \frac{\tan a - \tan b}{1+\tan a \tan b}

  7. \displaystyle\cot (a+b) = \frac{\cot a\cot b -1}{\cot b + \cot a}

  8. \displaystyle\cot (a-b) = \frac{\cot a\cot b +1}{\cot b - \cot a}

  • Multiples of \frac\pi2*

  1. \displaystyle\sin \left(\frac\pi2-x\right) = \cos x

  2. \displaystyle\sin \left(\frac\pi2+x\right) = -\cos x

  3. \displaystyle\sin (\pi-x) = \sin x

  4. \displaystyle\sin (\pi+x) = -\sin x

  5. \displaystyle\sin (2\pi-x) = -\sin x

  6. \displaystyle\cos \left(\frac\pi2-x\right) = \sin x

  7. \displaystyle\cos \left(\frac\pi2+x\right) = \sin x

  8. \displaystyle\cos (\pi-x) = -\cos x

  9. \displaystyle\cos (\pi+x) = -\cos x

  10. \displaystyle\cos (2\pi-x) = \cos x

  • Double Angle

  1. \sin 2x=2\sin x\cos x

  2. \begin{aligned}[t]\cos 2x &=2\cos^2 x - 1 \\ &= 1 - 2\sin^2 x\\ &= \cos^2 x-\sin^2 x\end{aligned}

  3. \displaystyle\tan 2x=\frac{2\tan x}{1+\tan^2x}

  • Triple Angle

  1. \sin 3x=3\sin x - 4\sin^3 x

  2. \cos 3x=4\cos^3 x - 3\cos x

  3. \displaystyle\tan 3x=\frac{3\tan x - \tan^3 x}{1-3\tan^2 x}

  • Sum/Difference of Sines and Cosines

  1. \displaystyle\cos a + \cos b=2\cos \left(\frac{a+b}2\right)\cos\left(\frac{a-b}2\right)

  2. \displaystyle\cos a - \cos b=-2\sin \left(\frac{a+b}2\right)\sin\left(\frac{a-b}2\right)

  3. \displaystyle\sin a + \sin b=2\sin \left(\frac{a+b}2\right)\cos\left(\frac{a-b}2\right)

  4. \displaystyle\sin a - \sin b=2\cos \left(\frac{a+b}2\right)\sin\left(\frac{a-b}2\right)

  • Product of Sines and Cosines (Inverse of 28-31)

  1. 2\cos a\cos b = \cos (a+b)+\cos(a-b)

  2. -2\sin a\sin b = \cos (a+b)-\cos(a-b)

  3. 2\sin a\cos b = \sin (a+b)+\sin(a-b)

  4. 2\cos a\sin b = \sin (a+b)-\sin(a-b)

  • Zero Values*

  1. \sin x = 0 \implies x=n\pi, n\in Z

  2. \displaystyle\cos x = 0 \implies x=(2n+1)\frac\pi2, n\in Z

  • Equal Values*
  1. \sin x = \sin y \implies x = n\pi + (-1)^ny
  2. \cos x = \cos y \implies x = 2n\pi \pm y
  3. \tan x = \tan y \implies x = n\pi + y

*These identities can be easily represented by the Unit Circle.
